# ASCII Art Generator 🎨

Un generador de arte ASCII moderno y fácil de usar que convierte imágenes y texto en arte ASCII. Construido con JavaScript vanilla y Tailwind CSS, ofrece una interfaz limpia y responsive con un diseño inspirado en aplicaciones SaaS modernas.

![ASCII Art Generator Demo](demo.gif)

## ✨ Características

### 🖼️ Conversión de Imagen a ASCII
- Soporte para múltiples formatos de imagen (PNG, JPG, GIF, etc.)
- Control ajustable de ancho (50-300 caracteres)
- Ajuste de contraste (0.5-2.0)
- Opción para invertir colores
- Múltiples conjuntos de caracteres ASCII:
  - Estándar (@#B8&o:*.)
  - Simple (#. )
  - Detallado (70+ caracteres)
  - Bloques Unicode
  - Bloques Minimal

### 📝 Conversión de Texto a ASCII
- Convierte texto en arte ASCII de 5x5
- Soporte para caracteres especiales:
  - Letras: A-Z + Ñ
  - Números: 0-9
  - Símbolos: ¿ ? ( ) / \ & % $ · # " ! ¡ ' = @ . ,
- Visualización previa en tiempo real

### 💾 Características Adicionales
- Interfaz de usuario intuitiva con pestañas
- Diseño responsive
- Descarga directa del arte ASCII generado
- Previsualización instantánea

## 🚀 Uso

1. **Conversión de Imagen**
   - Haz clic en "Subir Imagen"
   - Ajusta el ancho y contraste según necesites
   - Selecciona el conjunto de caracteres deseado
   - ¡Listo! Tu imagen se convertirá automáticamente

2. **Conversión de Texto**
   - Cambia a la pestaña "Texto a ASCII"
   - Escribe o pega tu texto
   - Haz clic en "Convertir"
   - El texto se transformará en arte ASCII

3. **Descarga**
   - Haz clic en "Descargar ASCII Art" para guardar tu creación
